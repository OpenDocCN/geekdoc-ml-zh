# 概念

> 原文：[`dafriedman97.github.io/mlbook/content/c2/concept.html`](https://dafriedman97.github.io/mlbook/content/c2/concept.html)

线性回归可以通过多种方式扩展以适应不同的建模需求。正则化回归通过惩罚回归系数的幅度来避免*过拟合*，这对于使用大量预测器的模型尤其有帮助。贝叶斯回归对回归系数施加先验分布，以便将现有关于这些参数的信念与从新数据中获得的信息相协调。最后，广义线性模型(GLMs)通过改变假设的误差结构，并允许目标变量的期望值是预测器的非线性函数，从而扩展了普通线性回归。这些扩展在本章中进行了详细描述、推导和演示。
