# 概念

> [`dafriedman97.github.io/mlbook/content/c3/concept.html`](https://dafriedman97.github.io/mlbook/content/c3/concept.html)

*分类器*是一种监督学习算法，试图识别观察值属于两个或更多组中的一个。换句话说，分类中的目标变量代表一个有限集合中的*类别*，而不是连续的数字。例如包括检测垃圾邮件或识别手写数字。

本章和下一章分别介绍了*判别分类*和*生成分类*。判别分类直接将观察值的类别成员资格建模为其输入变量的函数。生成分类则相反，将输入变量视为观察值类别的函数。它首先模型化观察值属于给定类别的先验概率，然后根据其类别计算观察值输入变量的概率，最后使用贝叶斯定理求解属于给定类别的后验概率。更多内容将在下一章中介绍。

本章迄今为止最常见的方法是逻辑回归。然而，这并非唯一的判别分类器。本章还介绍了另外两种：*感知器算法*和*费舍尔线性判别*。
