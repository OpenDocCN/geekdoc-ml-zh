# 3.2 股权授予

> [原文链接](https://huyenchip.com/ml-interviews-book/contents/3.1.2-equity-grants.html)

股权授予可以理解为承诺——公司承诺，如果您在那家公司待一定时间，将给予您一定数量的股权。当时间到了，您收到股权时，那部分股权就被称为已生效。

股权授予按照既定的时间表进行，公司内每个人的时间表都是相同的。最常见的两种时间表是：

+   在四年内平均分配。例如，如果您获得 100 股，每年年底将生效 25 股。

+   您的股权授予在第一年后四分之一生效，剩余部分在接下来的三年每月底平均生效。

您在一家公司待得越久，获得的股权就越多。您的新股权授予也将遵循一个授予时间表。例如，您加入公司时获得 100 股。第一年后，您有权获得 25 股。由于您表现良好，您在接下来的 4 年内还将额外获得 80 股。因此，在您的第二年结束时，您将有：25 股旧股 + 20 股新股 = 45 股。您在一家公司待得越久，剩余待生效的股权授予就越多。这个时间表是为了激励员工在一家公司待得更久。
