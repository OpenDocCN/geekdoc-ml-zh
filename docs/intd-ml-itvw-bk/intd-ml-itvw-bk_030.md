# 2.1.1.3 价值一致性究竟是什么？

> [原文链接](https://huyenchip.com/ml-interviews-book/contents/2.1.1.3-what-exactly-is-culture-fit.html)

公司寻找的一件事不是技能，有时甚至是一个有争议的来源：**文化契合度**。有些人甚至认为这是创造排他性文化的代理——管理者只雇佣那些看起来像他们、说话像他们、来自与他们相同背景的人。

一些公司已经将“文化契合度”这一术语替换为“价值一致性”等术语。契合度应该与价值观相一致，而不是生活方式，例如，你是否重视建设性的批评，而不是你是否每个周日下午都出去喝酒。

对于大多数大型公司来说，因为他们的文化已经确立，一位新员工不太可能改变办公室的动态，文化契合度最终归结为你是否是人们愿意与之共事的人（例如，你不是个混蛋，你不是防御性的，你是一个团队玩家）。对于小型组织来说，文化契合度更为重要，因为公司希望人们分享他们的使命、动力、道德观和工作习惯。

价值一致性也关乎你评估这家公司是否是你想要工作的公司。你相信他们的使命和愿景吗？他们目前的员工是不是你希望与之共事的人？公司是否会为你提供一个良好的成长环境？一位候选人告诉我，他在被邀请参加公司俯卧撑比赛后拒绝了工作邀请。他觉得一个如此重视充满睾酮活动的文化并不适合他。
