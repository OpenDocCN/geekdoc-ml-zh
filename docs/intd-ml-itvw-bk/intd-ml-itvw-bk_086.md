# Chapter 7\. Machine learning workflows

> 原文：[`huyenchip.com/ml-interviews-book/contents/chapter-7.-machine-learning-workflows.html`](https://huyenchip.com/ml-interviews-book/contents/chapter-7.-machine-learning-workflows.html)

尽管深度学习似乎成为了研究社区讨论的焦点，但大多数现实世界的问题仍然是通过传统的机器学习算法来解决的，包括 k-最近邻和 XGBoost。在本章中，我们将涵盖理解机器学习算法所必需的基本原理，以及你可能在日常工作和面试中找到有用的非深度学习算法。
