# 6.3 数据

> 原文：[`huyenchip.com/ml-interviews-book/contents/6.3-data.html`](https://huyenchip.com/ml-interviews-book/contents/6.3-data.html)

在学术或研究环境中，你很可能只处理干净、易于获取的数据集，因此你可以花大部分时间在建模上。在生产环境中，你可能会花大部分时间在数据处理管道上。能够管理、处理和监控数据的能力会使你成为潜在雇主的吸引点。

在你的面试中，可能会被问到一些评估你与数据打交道舒适度的问题。从高层次来说，你应该熟悉读取、写入和序列化不同类型的数据。你应该有一个用于数据框操作的常用库：`pandas`在通用数据应用中很受欢迎，如果你需要 GPU 兼容的选项，`dask`是一个不错的选择。你应该至少熟悉一个可视化库，如`seaborn`、`matplotlib`、`Tableau`或`ggplot`。

如果你想要处理大数据，熟悉分布式数据管理系统，如 Spark 和 Hadoop，是有益的。

除了 Python 之外，SQL 对于所有需要持久化数据库的应用仍然无处不在，尽管 R 语言不是最吸引人的语言，但它对于快速数据分析来说很方便。
