# 1.1.1 在研究工作中与在生产工作中

> 原文：[`huyenchip.com/ml-interviews-book/contents/1.1.1-working-in-research-vs.-workingin-production.html`](https://huyenchip.com/ml-interviews-book/contents/1.1.1-working-in-research-vs.-workingin-production.html)

*我使用“研究 vs. 生产”而不是“学术界 vs. 工业”，因为尽管学术界主要关注研究，但研究并不主要在学术界进行。事实上，如今机器学习研究是由大公司引领的。参见**1.1.2 研究**获取更多详情。*

你可能想要弄清楚的第一个问题是，你是否想在研究或生产中工作。它们的工作描述、要求、招聘流程和薪酬差异很大。

研究的目标是找到基本问题的答案并扩展理论知识体系。一个研究项目通常涉及使用科学方法来验证一个假设或理论是否正确，而不必担心结果的实际性。

生产的目标是创建或增强产品。产品可以是商品（例如汽车）、服务（例如共享乘车服务）、流程（例如检测交易是否欺诈）或商业洞察（例如“为了最大化利润，我们应该将价格提高 10%”）。

一个研究项目不需要用户，但产品需要。为了使产品有用，它除了性能之外还有许多其他要求，例如推理延迟、可解释性（对用户和开发者而言）、公平性（对所有用户子组而言）、对变化环境的适应性。生产团队的大部分工作可能是确保这些其他要求。

上述定义当然是最基本的。截至 2021 年，机器学习中的“研究”和“生产”仍然是一个热烈的辩论话题。造成这种模糊性的一个原因是，具有明显有用性的新想法往往吸引更多研究人员，而解决实际问题通常需要提出新想法。

关于研究中的机器学习与生产中的机器学习的更多差异，请参阅[斯坦福大学 CS 329S，第一部分：理解机器学习生产](https://docs.google.com/document/d/1VuofeF5okBATz1F7HRQmOgi5Jc4bmUiHMYjRqwF-29s/edit?usp=sharing)。

作为候选人，如果你对两者都不熟悉，并且不确定是否想在研究或生产中寻找角色，后者可能是一条更平稳的道路。涉及生产的角色比涉及研究的角色要多得多。

^(3)：一个例子是关于 GPT-3 是否属于研究的争论。当《语言模型是少样本学习者》（OpenAI，2020）在 NeurIPS 获得最佳论文奖时，许多研究人员感到不满，因为[他们没有将其视为研究](https://twitter.com/chipro/status/1337514435800256512)。
