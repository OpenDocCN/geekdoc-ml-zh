# 1.1.2.1 研究与应用研究

> [原文链接](https://huyenchip.com/ml-interviews-book/contents/1.1.2.1-research-vs.-applied-research.html)

在一些公司，你可能会遇到涉及应用研究（**applied research**）的职位。应用研究位于研究和生产之间，但更接近研究而非生产。应用研究涉及寻找解决实际问题的方案，但不涉及在真实生产环境中实施这些方案。

应用研究者是研究者。他们提出新的假设和论点，并验证它们。然而，由于他们的假设和论点涉及实际问题，他们也需要理解这些问题。在行业术语中，他们需要具备**专业知识**。

在机器学习中，一个研究项目的例子可能是开发一种用于计算机视觉的无监督迁移学习方法，并在标准学术数据集上进行实验。一个应用研究项目的例子可能是开发技术，使该方法能够在特定行业的真实世界问题中工作，例如医疗保健。因此，从事这个应用研究项目的人将需要具备机器学习和医疗保健两方面的专业知识。
