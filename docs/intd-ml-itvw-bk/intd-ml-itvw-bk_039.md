# 2.2.4 实习生面试与全职职位面试

> 原文：[`huyenchip.com/ml-interviews-book/contents/2.2.4-interviews-for-internships-vs.-for-full-time-positions.html`](https://huyenchip.com/ml-interviews-book/contents/2.2.4-interviews-for-internships-vs.-for-full-time-positions.html)

通常，面试实习生然后获得全职员工的回头 offer 比直接面试全职职位要容易得多。面试过程是预测你在工作中表现如何的代理。如果你已经与一个团队实习过，他们了解你的能力和适应性，因此可能更倾向于你而不是一个未知候选人。

在主要科技公司中，实习项目存在是为了提供一个稳定、可靠的全职人才来源。Facebook 的平均实习生月薪为 8,000 美元，几乎是普通美国全职员工的两倍^(43)，而 Facebook 甚至不是出价最高的公司^(44)。除非它能抵消未来的招聘成本，否则很难证明这种薪资的合理性。

实习生的面试过程相对简单，因为实习的承诺性较低。如果公司不喜欢一个实习生，那个实习生将在三个月后离开。但如果不满意全职员工，解雇那个人是昂贵的。全职职位的数量受到严格的员工人数限制，但实习生的数量通常不受限制。即使公司冻结招聘流程，例如为了削减成本，他们也可能仍然会招聘实习生。

如果你的实习没有出太大差错，并且公司正在招聘，你很可能会得到一份从实习生到全职员工的 offer。在 NVIDIA，为新毕业生提供的全职 offer 中的大多数都给了他们的实习生。Rachelle Gupta，前谷歌和 GitHub 的招聘人员，在 Quora 上的一个回答中写道：

“*实习生转换率的范围在 60% 以上，在顶级科技公司中甚至可以达到 90% 以下。我所说的转换是指‘有资格转为全职员工’而不是仅仅有资格获得明年的实习机会。在谷歌，当一个团队获得实习机会时，它附带一个全职职位，实习生可以转换为全职员工*。”^(45)”

如果你是一名学生，开始寻找实习永远不会太早。高中学生实习并不罕见。然而，如果你已经过了那个阶段，不要担心。这本书旨在帮助你完成这个过程。
