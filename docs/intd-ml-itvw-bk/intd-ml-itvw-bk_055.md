# 3.3 奖金

> 原文：[`huyenchip.com/ml-interviews-book/contents/3.1.3-bonuses.html`](https://huyenchip.com/ml-interviews-book/contents/3.1.3-bonuses.html)

奖金种类繁多，但最常见的两种是入职奖金和年度奖金。初级职位的入职奖金可以从 0 美元到超过 10 万美元不等。对于更高级别的职位，奖金可能没有上限。我见过入职奖金大约在五十万美元左右的。

年度奖金通常占基本工资的 10%到 20%。然而，像 Netflix 这样的公司不相信奖金，更愿意将奖金包含在基本工资中，因为他们认为“基本工资 20 万美元加上 15%的绩效奖金”对候选人来说不如“基本工资 23 万美元”有吸引力。
