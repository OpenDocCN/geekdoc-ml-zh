# B. 建立你的网络

> [原文链接](https://huyenchip.com/ml-interviews-book/contents/b.-building-your-network.html)

如果你之前认为拥有一个网络关系并不重要，我希望这本书能改变你的看法。拥有正确的网络关系可以为你打开新的机会，让你更接近你的梦想工作。

最好的联系是那些你曾经学习或工作过的人。选择性地上学或加入一家选择性的公司的主要价值在于网络。你从麻省理工学院或斯坦福大学的同学最终可能会加入 FAAAM，而你从 FAAAM 的朋友最终可能会创立自己的公司。

如果网络不是你教育中固有的功能，你就得付出努力。我理解，你讨厌社交。与陌生人交谈的奖励函数是随机的。偶尔，你会遇到一个能与你合得来的人。但大多数时候，这很无聊。自从大学一年级以来，我就没有参加过任何社交活动。为什么要在社交活动中站在角落里，而你可以在沙发上坐着，吃着披萨，浏览搞笑图片呢？

但社交不一定要关于遇见随机的陌生人并强迫建立联系。它可以是你找到与你兴趣相投的人，并从他们那里学习新事物。寻找那些做过你感兴趣的事情的人，并尝试就他们的工作进行建设性的讨论。

当我刚开始学习机器学习时，我尝试重新实现论文。当我遇到我不理解或我认为可以做得不同的地方时，我会给作者发邮件。大多数人都会对关于他们工作的讨论持开放态度——每个人都希望自己的工作更受欢迎。在这个过程中，我认识了很多朋友。

有时候，来回发邮件并不有效，我会请人们见面喝咖啡或打电话。他们有时不回应或拒绝。我也收到过这样的消息。只要发件人尊重，他们的消息就会让我高兴（有人认可我的工作！）我会尽量为他们腾出时间。如果我不回应或拒绝，那是因为我忙或忘记了。

发布你的作品，以便对它感兴趣的人可以联系你。我喜欢做一些副项目来满足自己的好奇心，但它们也有助于让我的名字为人所知。我经常收到这样的邮件：“你对 X 的分析真的很酷。你认为它能在 Y 上工作吗？”或者“我可以在我项目 Z 中使用它吗？”有时，我们会找到合作的方式，有时，他们成为了我的朋友。我不止一次收到来自我钦佩但不敢联系的人的邮件。

如果你注意到了如何使用社交网络，它们会非常有帮助。机器学习领域的人士在 Twitter 上出奇地活跃。我关注那些我尊重的人，并在我有话要说时参与他们的公开对话。我发布我关心的事情，以便吸引只有与我兴趣相投的人。如果有人发布了我认为有帮助的内容，或者对我的帖子做出了深思熟虑的回应，我会发一条简短的消息感谢他们的努力。

一流的会议，如 NeurIPS、ICLR、ICML、CVPR、ACL、SIGGRAPH，也是结识人脉的好地方。很多人可能不同意我的看法，但我相信，由于大多数一流会议都会在线上发布他们的演讲和论文，参加会议的主要价值在于结识其他与会者。你应该提前查看会议日程和已接受论文列表，看看谁会参加，并主动联系你想见的人。在会议期间，尽量与尽可能多的人谈论他们的工作（以及你的工作）。

如果你能在主要会议或其中一个研讨会中展示你的工作，你就能从会议中获得更多。当然，总会有例外，但一般来说，将论文提交给研讨会比提交给主要会议更容易。主要会议和研讨会通常为需要经济援助的已接受作者提供旅行补助金。

建立人脉是一个终身的过程，不是你可以在周末“破解”的事情。要开放地结识新人，发现他们知道而你不知道的东西。不要因为你想和他们交朋友而接近某人——你不能强迫友谊，而且这有点令人毛骨悚然。如果你有想和他们讨论的事情，就接近他们。如果双方都有共同的兴趣，友谊自然会自然产生。

当你准备好寻找机会时，不要害怕让你的网络中相关的人知道。大多数人都会很高兴为优秀的候选人提供推荐。他们希望与聪明的人一起工作，如果他们的候选人被录用，他们也会得到推荐奖金。

你也可以向网络外的人寻求推荐。我不止一次为那些给我发冷邮件请求推荐的人推荐候选人。工程师通常比招聘人员更擅长判断一个人的技术能力。如果你缺乏招聘人员寻找的信号，你可能更有可能给其他工程师留下深刻印象。

最好的求职策略是建立一个难以忽视的简历。撰写深入的技术博客文章或论文，为开源项目做出贡献，参加黑客马拉松，参加编码挑战赛，最重要的是，发布你的工作，让世界知道你是一位多么出色的工程师。
