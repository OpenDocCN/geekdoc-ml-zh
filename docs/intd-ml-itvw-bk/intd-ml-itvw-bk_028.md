# 2.1.1.1 技术技能

> 原文：[`huyenchip.com/ml-interviews-book/contents/2.1.1.1-technical-skills.html`](https://huyenchip.com/ml-interviews-book/contents/2.1.1.1-technical-skills.html)

1.  **软件工程**。由于机器学习模型通常需要大量的工程来训练和部署，因此理解工程原理非常重要。与机器学习更相关的计算机科学方面包括算法、数据结构、时间/空间复杂性和可扩展性。你应该熟悉常规工具：Python、Jupyter Notebook 或 Google Colab、NumPy、scikit-learn^(19)以及深度学习框架。了解至少一种性能导向的语言，如 C++或 Go，可能会很有帮助。[BestPracticer](https://bestpracticer.com/topic/engineering-1521158538386x966527170501649400)列出了不同级别所需的不同工程技能。

1.  **数据清洗、分析和可视化**。数据处理在机器学习教育中非常重要，但往往被忽视。如果候选人知道如何收集、探索、清洗数据，以及如何创建训练数据集，这将是一个巨大的加分项。你应该熟悉数据框操作（pandas, dask）和数据可视化（seaborn, altair, matplotlib 等）。SQL 在关系型数据库中很流行，而 R 在数据分析中很流行。熟悉如 Spark 和 Hadoop 这样的分布式工具包也非常有用。

1.  **机器学习知识**。你应该对机器学习有更深入的理解，而不仅仅是引用流行词汇。理想情况下，你应该能够解释你做出的每一个架构选择。如果你只是克隆现有的开源实现，并且它在你的数据上运行得很好，你可能不需要这种理解。但是，模型很少能够完美运行，所以你需要这种理解来评估潜在解决方案和调试你的模型。

1.  **特定领域知识**。你应该具备与你面试的公司产品相关的知识。如果你在自动驾驶领域，你可能会被期望了解计算机视觉技术以及计算机视觉任务，如目标检测、图像分割和运动分析。如果公司构建语音识别系统，你应该了解梅尔滤波器组特征、CTC 损失以及语音识别任务的常见基准数据集。

* * *

^(19)：截至 2019 年，scikit-learn 与 TensorFlow 一样受欢迎。
