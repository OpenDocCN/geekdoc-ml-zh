# 2.3.2 向面试官提问的问题

> [原文链接](https://huyenchip.com/ml-interviews-book/contents/2.3.2-questions-to-ask-your-interviewers.html)

你通常在每个面试的结尾有 5-10 分钟的时间，在行为面试环节则有更多时间来提问。有时候，你可以从一个人的问题中了解到比他们的答案更多的信息。

面试过程是双向的——不仅公司评估你是否适合，你也在评估你是否想为该公司工作。你应该利用你拥有的每一个机会来提问，了解公司——他们的使命、愿景、价值观、竞争对手、未来计划、他们面临的挑战、你应该了解的政策、内部层级和现有的公司政治。

你应该了解你将要面试的团队：团队构成和动态，团队活动，管理风格，他们希望加入团队的人的类型。你也应该尝试了解你预期要参与的项目以及你的表现将如何被评估。

如果你关心你工作的可见性——这对于职业生涯早期的人来说尤为重要——你应该询问公司的出版政策：他们是否发布论文和开源代码？如果一个公司根本不发布任何内容，你加入后就会消失——外界根本不知道你在做什么。

最近，在主要科技公司中出现了员工罢工，以抗议其雇主与政府某些部门的合作。如果你关心这个问题，你绝对应该向你的潜在雇主询问他们与政府合作的态度。

你的面试官的职业视角是你加入公司后可能面临的情况的一个很好的指示，所以你应该尝试了解他们做什么以及为什么/如何这样做。他们为什么选择这家公司？它与他们之前雇主相比如何？他们认为自己的工作有什么挑战？他们在选择要做什么工作时有多少自由？

如果你需要更多的问题来提问，Julia Evans 有一个相当不错的[列表](https://jvns.ca/blog/2013/12/30/questions-im-asking-in-interviews/)。

> ⚠永远不要向你的面试官询问薪酬⚠
> 
> 除非面试官明确提出这个话题。一些招聘经理认为这是一个红旗，因为它表明候选人只关心金钱，一旦有更好的工作机会就会跳槽。
