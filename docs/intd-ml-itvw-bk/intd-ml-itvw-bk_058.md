# 3.2.1 薪酬期望

> 原文：[`huyenchip.com/ml-interviews-book/contents/3.2.1-compensation-expectations.html`](https://huyenchip.com/ml-interviews-book/contents/3.2.1-compensation-expectations.html)

候选人害怕的问题是：“你的薪酬期望是多少？” 一种回答方式是让招聘人员先给出一个数字。以下是一个如何回答这个问题的例子。

*“我对这个机会感到兴奋，我相信如果这是一个好的匹配，我们将就薪酬细节达成协议。我想先了解这个职位包含的内容，以便有一个现实的期望。如果你能和我分享这个职位可以期待的薪酬范围，那对我也会有帮助。”*

这个答案有效，因为它表明你准备好谈判，但你同时也希望让它成功。

询问招聘人员给出一个数字的一个问题是，谈判将围绕它展开。如果他们给出的数字太低，你可能会觉得很难显著提高它。

另一种回答方式是给出一个范围，但不做出承诺。想出一个你想要的数字，加上大约 20%，然后将其作为下限。例如，如果你目标是每年 20 万美元，你可以这样说。

*“我的范围是灵活的。然而，我希望根据我的经验、我独特的技能组合以及这份工作所包含的内容得到公平的薪酬。我的理解是，在这个领域的这个职位，你可以期望每年有 22 万到 24 万美元。”*

一旦你给出了一个数字，它只能下降，不能上升，所以请确保给出一个你以后不会后悔的数字。

想要更多谈判技巧，请查看 [15 条谈判工作报价规则](https://hbr.org/2014/04/15-rules-for-negotiating-a-job-offer)（哈佛商业评论，2014）和 [10 条谈判工作报价规则](https://haseebq.com/my-ten-rules-for-negotiating-a-job-offer/)（哈希布·库雷希，2016）。
