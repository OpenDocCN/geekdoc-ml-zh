# 目标受众

> [原文链接](https://huyenchip.com/ml-interviews-book/contents/0-target-audience.html)

如果你拿起这本书是因为你对在 2020 年代的关键新兴技术之一工作感兴趣，但不确定从哪里开始，你就在正确的位置。无论你想要成为一名 ML 工程师、平台工程师、研究科学家，还是你想要做机器学习但还不知道这些头衔之间的区别，我希望这本书能给你一些有用的建议。

这本书更侧重于涉及机器学习生产的角色，而不是研究，这并不是因为我认为生产更重要。随着越来越少的公司能够承担纯研究，而越来越多的公司希望采用机器学习，涉及生产的角色将远远多于研究，这种情况已经出现，并将继续存在。

这本书主要针对两组候选人群体：

1.  寻找第一份全职工作的应届毕业生。

1.  希望转型进入机器学习领域的软件工程师和数据科学家。

我想象这本书的大多数读者都来自计算机科学背景。书中关于问题的第二部分相对较为技术性。然而，随着机器学习在更多行业中的应用——医疗保健、农业、卡车运输、时尚，等等——这个领域需要更多具有多样化兴趣的人才。如果你对机器学习感兴趣，但因为没有工程学位而犹豫不决，我强烈建议你探索它。这本书，尤其是第一部分，可能满足你的一些需求。毕竟，我在做了近十年的作家工作之后才开始对矩阵运算感兴趣。
