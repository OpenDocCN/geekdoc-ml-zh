# 符号表示法

> 原文：[`huyenchip.com/ml-interviews-book/contents/notation.html`](https://huyenchip.com/ml-interviews-book/contents/notation.html)

如果某些字符似乎缺失，那是因为 MathJax 没有正确加载。刷新页面应该可以解决这个问题。

+   本书中的所有向量都是维度为 n 的列向量，其中 n 是该向量中元素的数量。

+   上标用于表示样本的索引。数据集中的第 k 个元素表示为 k。

+   下标用于表示维度。一个 n×m 矩阵的下标表示其第一行。
