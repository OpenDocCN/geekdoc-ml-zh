# 2.3.1.2 兴趣

> 原文：[`huyenchip.com/ml-interviews-book/contents/2.3.1.2-interests.html`](https://huyenchip.com/ml-interviews-book/contents/2.3.1.2-interests.html)

对公司来说，让员工对他们的工作充满热情是符合财务利益的。他们想确保你的兴趣与公司一致。以下是一些例子问题：

+   你为什么对机器学习感兴趣？

+   你对这个职位有什么兴趣？

+   你希望在下一份工作中学到什么？

+   在你当前的角色中，对于哪些类型的问题你是首选的人？如果我问你团队中的其他人这个问题，他们会说什么？

Joshua Levy，AI 力量 SaaS 公司 BloomReach 的创始人工程师，表示他最喜欢的兴趣问题是：“*回想一下你兴奋地去上班的时期。是什么让你感到兴奋？*”是你要解决的问题、你能产生的影响、你能接触到的技术、你合作的同事、你的学习曲线，还是其他什么？

许多面试官的一个常见问题是“*解释一篇你真正喜欢的论文*。”这个答案不仅显示了你对哪些问题的兴趣，还展示了你愿意走多远去理解一个问题，以及你如何有效地传达技术概念。无论你选择哪篇论文，只要你足够理解它，能够谈论它即可。在面试之前，选择任何一篇论文，彻底阅读并亲自实现它，以理解所有的细微之处。

我个人的最爱是：“*你做过什么让你最自豪的事情？*”这个问题的答案甚至不必是技术性的。我曾经告诉一个面试官，我最自豪的是我在高中时创立的非营利组织，旨在鼓励年轻人探索他们的舒适区之外。我得到了一份工作机会。

面试官想要了解你是谁：是什么激励你做到最好，你如何处理障碍。雇主希望你在他们的公司克服障碍，做到最好。公司希望有激情的人，所以谈谈那些能让你眼中闪烁火花的事情。
