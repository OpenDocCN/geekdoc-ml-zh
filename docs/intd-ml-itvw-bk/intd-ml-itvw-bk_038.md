# 2.2.3 大公司与小公司面试对比

> 原文：[`huyenchip.com/ml-interviews-book/contents/2.2.3-interviews-at-big-companies-vs.-at-small-companies.html`](https://huyenchip.com/ml-interviews-book/contents/2.2.3-interviews-at-big-companies-vs.-at-small-companies.html)

在小公司招聘与在大公司招聘非常不同。一个原因是大公司需要招聘大量人员，因此他们需要标准化他们的流程。小公司则是在过程中不断优化流程。另一个原因是大公司偶尔可以承担错误的招聘，而几个错误的招聘可能会把小公司拖垮。

因此，小公司的流程可以适应每个角色和每个候选人。大公司的流程可能比较僵化和官僚，可能包括与角色或候选人无关的问题。大公司的标准化也意味着流程更容易被破解——充分的准备可以显著提高你的胜算。

另一个重要的区别是，大公司可以承担聘请专家的费用，这些专家只在很小的领域内很出色。初创公司不可预测且不断变化，因此他们可能更关心你是否能解决他们最紧迫的需求，而不是你擅长做什么。
