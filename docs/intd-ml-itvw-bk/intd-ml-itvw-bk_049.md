# 2.5 时间线

> 原文：[`huyenchip.com/ml-interviews-book/contents/2.5-timeline.html`](https://huyenchip.com/ml-interviews-book/contents/2.5-timeline.html)

招聘速度因公司而异，差异很大。在时间轴的较长端，谷歌和 DeepMind 的处理过程通常需要六周到三个月。这个过程之所以长，部分原因是因为他们持续不断地涌入候选人，部分原因是因为他们在公司层面做出招聘决定。谷歌的招聘委员会每两周都会开会，审查所有通过面试过程的候选人，并决定要雇佣谁。如果你已经进入招聘委员会，你的机会看起来很好。

在时间轴的较短端，我们有初创公司和具有扁平化组织结构的大公司。如果初创公司真的想要你，整个过程可能只需要几天。像 NVIDIA 和 Netflix 这样的公司反应迅速，因为他们在团队层面做出招聘决定。团队经理可以当场做出招聘决定。我在 NVIDIA 的实习招聘过程不到一周就完成了。我在下午和我的经理谈了转正的事情，第二天早上就收到了我的录用通知。

其他公司则分布在中间位置。时间线在很大程度上取决于你面试的可用性、公司面试位置的可用性、他们对你有多大的需求以及他们需要你的程度。如果你还没有被告知预期的期限，你应该向你的招聘人员询问，并告知他们你任何的时间限制。

如果你面试后至少过去了一周还没有收到招聘人员的消息，你可以发送一个简短而礼貌的跟进信息。让他们知道你对公司很感兴趣，并希望了解任何更新，因为你需要考虑其他机会。
