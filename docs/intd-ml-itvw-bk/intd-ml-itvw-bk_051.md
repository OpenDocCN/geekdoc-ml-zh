# 第三章\. 接到 offer 之后

> 原文：[`huyenchip.com/ml-interviews-book/contents/chapter-3.-after-an-offer.html`](https://huyenchip.com/ml-interviews-book/contents/chapter-3.-after-an-offer.html)

拒绝之后生活并没有结束。接到 offer 之后也不会结束。你可能想要协商你的 offer，考虑其他机会，思考在公司或离职后的职业前景。

在加入公司之前就考虑离职似乎很荒谬，但技术行业以 13.2%的离职率在所有商业领域中最高。亚马逊的平均任期是一年，谷歌是 1.1 年^(50)。你可能会发现自己比想象中更快地再次成为候选人。

在本章中，我们将讨论在考虑你的选择时可能有用的薪酬细节和职业发展。

* * *

^(50): [技术专业人士的真实问题：高离职率](https://www.forbes.com/sites/sap/2020/07/09/how-to-better-meet-your-customer-needs-and-do-so-more-quickly/). Forbes, 2020.
