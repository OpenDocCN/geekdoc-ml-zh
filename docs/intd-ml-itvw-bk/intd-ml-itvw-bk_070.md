# 第二部分：问题

> 原文：[`huyenchip.com/ml-interviews-book/contents/part-ii.-questions.html`](https://huyenchip.com/ml-interviews-book/contents/part-ii.-questions.html)

这一部分包含超过 200 个有或无确定性答案的问题。这类问题旨在测试你对机器学习概念的理解。在一个小时的面试中，你可以覆盖 10 到 15 个这样的问题。我根据三个难度级别对问题进行排序：

+   **[E]**asy - 基础的，每个人都应该知道。

+   **[M]**edium - 理解概念不仅限于定义的人应该知道。

+   **[H]**ard - 有经验的人应该知道。

一些知识性问题被认为是糟糕的面试问题，尤其是那些可以轻易查阅的定义问题。例如，要求某人解释主成分分析（PCA）可以用来评估他们对 PCA 的记忆，而不是对 PCA 的理解。然而，一些糟糕的面试问题在面试练习中仍然可以变成好问题，因此我包括了一些定义问题，以提醒读者某些概念的重要性。

技术会兴衰，但基本挑战保持不变。而不是要求候选人写出某些技术的复杂方程式，这本书专注于最初引发这些技术的挑战。本节的大部分问题都是关于为什么某些东西很重要以及它是如何工作的。

> 🌳 **提示：`定义`问题的策略** 🌳
> 
> 当被要求给出某个技术定义或解释该技术时，始终从该技术的动机开始。例如，如果被要求解释循环神经网络中的 LSTM，你应该首先提出正常 RNN 中出现的哪些问题，以及 LSTM 是如何解决这些问题的。
