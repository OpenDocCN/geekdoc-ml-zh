# 第四章：从哪里开始

> 原文：[`huyenchip.com/ml-interviews-book/contents/chapter-4.-where-to-start.html`](https://huyenchip.com/ml-interviews-book/contents/chapter-4.-where-to-start.html)

在一个更美好的世界里，我们不应该需要为面试做准备。在一个理想的世界里，甚至不应该有面试。不幸的是，在我们这个世界上，为了我们的能力能够得到积极的评估，我们需要做准备。

这种准备是一个终身的过程，应该在开始申请任何工作之前就开始。它包括建立你的作品集、经验、技能组合和人际关系网。只有在你简历上有好东西可以展示时，你才能写出一个好的简历。

如果你现在才想到想要找工作，还不算太晚。无论你现在做什么工作，都要继续做准备，这样你才能在下次求职时展现出最好的自己。

本节涵盖了加强你的申请的资源。它们包括免费的在线课程、书籍、文章、你需要熟悉的工具等。你可以投入的准备程度取决于你的时间表。
