# 2.1.1 公司希望从候选人那里得到什么

> [`huyenchip.com/ml-interviews-book/contents/2.1.1-what-companies-want-from-candidates.html`](https://huyenchip.com/ml-interviews-book/contents/2.1.1-what-companies-want-from-candidates.html)

面试流程的目标是让公司评估：

1.  你是否具备完成这项工作所需的知识和技能

1.  是否能为你提供一个适合执行该任务的环境。

公司将评估你的技术和非技术技能。
