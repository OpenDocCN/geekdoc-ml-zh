# 3.1 补偿方案

> 原文：[`huyenchip.com/ml-interviews-book/contents/3.1-compensation-package.html`](https://huyenchip.com/ml-interviews-book/contents/3.1-compensation-package.html)

补偿有两种类型：直接补偿和间接补偿。对于大多数科技公司，直接福利包括三个主要组成部分：

+   基础工资（现金）

+   股权授予（股票、期权）

+   奖金

间接福利更加多样化，并且可能非常慷慨，但通常在主要科技公司中。以下是一些例子。

+   医疗保险

+   401(k) 匹配

+   供应餐点、小吃和饮料

+   健康补贴用于健身房会员费、运动装备报销、按摩、物理治疗、冥想

+   交通、电话、电话计划的补贴

+   租车、航班、电子产品、活动的企业折扣

+   学生贷款偿还

+   教育援助计划，如学费报销

+   会议出席协助

+   有薪休假
