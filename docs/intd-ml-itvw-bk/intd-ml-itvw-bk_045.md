# 2.3.1.4 个性

> [`huyenchip.com/ml-interviews-book/contents/2.3.1.4-personality.html`](https://huyenchip.com/ml-interviews-book/contents/2.3.1.4-personality.html)

公司想了解你，你的个性，你的毅力，你的优点和缺点。直接询问候选人的优点和缺点通常会导致结果不佳——大多数人都不愿意承认自己的缺点，也太害羞去夸耀自己的优点。因此，面试官可能会以不同的方式提出问题。

这里有一些面试官可能会提出的问题。

+   你的同事会如何描述你？

+   如果你正在面试一个潜在队友，你会寻找哪些特质？

+   如果你需要一个伙伴来解决编程问题，你会如何描述你理想的伙伴？

+   如果你可以回到五年前，你会学习什么技能？

+   请告诉我一次你面对看似无法克服的挑战以及你是如何克服它的经历。

高级面试官往往有一个他们认为能揭示一个人最多信息的固定问题。在他的书《从零到一》中，彼得·蒂尔透露，他向所有面试的人都问过：“*有什么重要的真理是很少有人与你意见一致的？*”这个问题很难回答，因为它首先要求你了解大多数人同意什么，能够独立思考，并且有足够的自信来表达它。

做你自己，但不要成为一个混蛋。硅谷一直以容忍杰出的混蛋而闻名——这些混蛋是表现优异但粗鲁且令人不愉快的同事，这导致了科技行业所谓的有毒文化。然而，近年来，包括 Netflix 在内的许多科技公司都承诺不雇佣混蛋，无论他们多么有才华。
