# 2.3.1.3 沟通

> 原文：[`huyenchip.com/ml-interviews-book/contents/2.3.1.3-communication.html`](https://huyenchip.com/ml-interviews-book/contents/2.3.1.3-communication.html)

沟通可以说是工作中最重要的软技能。如果团队成员不能或不愿意相互沟通，团队合作是不可能的。每家公司都会想评估你沟通的能力，以及你是否能够适应公司的沟通风格。例如，如果一家公司非常重视日常状态更新，而你更喜欢消失几天专注于一个项目，只在完成时出现，那么可能并不适合。

你的沟通风格和技能在整个过程中都会被评估，从你的求职信，如何回复他们的邮件，到回答每个面试问题的方式。公司也可能直接询问你如何沟通的问题。以下是一些需要思考的问题：

+   请告诉我一次你必须给出建设性批评的经历。

+   请告诉我一次你收到批评反馈的经历。

+   你是否曾与你的经理意见不合？你是如何处理的？

+   你的同事会如何描述你的沟通风格？

+   你参与过的最模糊的项目是什么？你是如何处理的？

+   什么样的管理风格能激发你最好的表现？
