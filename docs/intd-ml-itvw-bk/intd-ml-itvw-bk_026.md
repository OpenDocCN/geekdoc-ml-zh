# 2.1 理解面试官的心态

> 原文：[`huyenchip.com/ml-interviews-book/contents/2.1-understanding-the-interviewers’-mindset.html`](https://huyenchip.com/ml-interviews-book/contents/2.1-understanding-the-interviewers’-mindset.html)

要理解面试流程，重要的是要从雇主的视角来看待。不仅候选人讨厌招聘过程，雇主也讨厌。这对公司来说很昂贵，对招聘经理来说很紧张，对面试官来说很无聊。

当一些知名组织被简历淹没时，不太知名的公司却在努力吸引人才。我经常听到小公司说，几乎不可能与科技巨头提供的报价竞争。在花费数周时间全力以赴追求候选人后，公司提出报价，却发现 FAAAM 已经出价更高。公司通常会聘请人才代理机构，这些机构可能会为招聘人员收取相当于第一年工资 20-30%的费用，对于一个初级工程师的招聘来说，这很容易转化为 5 万美元。

在硅谷，人才竞争尤其激烈，因为每人的公司数量众多，这使得候选人的机会更大。招聘人员，即使是那些每年收到数百万份简历的公司（如谷歌^(17）的招聘人员，也会积极追求潜在的候选人，即使这些候选人并不在寻找工作。2018 年接受新工作的大多数人并不是在寻找工作^(18)。

一些候选人对招聘人员的主动联系表示轻微的不满。这种态度通常是不恰当的，因为招聘人员是你的最大盟友：他们努力让你被雇佣。作为候选人，你希望有足够的曝光度，以便招聘人员能够联系到你。

每家公司都说他们想要招聘最优秀的人才。这并不真实。**公司想要招聘的是那些在时间和金钱限制内能合理完成工作的最优秀人才**。如果需要一个月和 1 万美元才能找到能完成 93%工作的候选人 A，而需要六个月和 5 万美元才能找到能完成 96%工作的候选人 B，大多数公司会选择候选人 A。即使是最优秀的工程师，如果他们没有突出招聘人员看重的能力，也可能被忽视。

你可能会认为，当公司招聘时，他们确切地知道他们想要新员工做什么。除非是一个有固定任务的成熟团队，否则招聘经理很少能完美清晰地预测需要完成哪些任务或需要哪些技能。有时，公司甚至不能确定他们是否需要那个人。Y Combinator 的董事长兼 OpenAI 的联合主席 Sam Altman 建议公司，在开始时，“**你应该只在极度需要的时候才招聘**。”(http://startupclass.samaltman.com/courses/lec02/)

然而，由于招聘竞争激烈且耗时，公司无法等到绝望时才行动。绝望的招聘很可能是糟糕的招聘。Amplify Partners 专注于 AI 的合伙人 Sarah Catanzaro 建议她的投资公司在他们有 50%的把握需要某人时开始招聘。

想象一下一家刚刚筹集了几百万美元的初创公司，他们决定将他们的日志转化为有用的功能。他们认为机器学习可以帮助他们，但不知道该如何实现。当招聘人员要求他们提供职位描述时，他们草草地列出了一系列他们认为可能必要的通用机器学习相关技能和经验。例如，“5 年经验”或“相关领域的学位”这样的要求是任意的，可能会阻止他们雇佣到合适的候选人。

> 🌳 **提示** 🌳
> 
> 职位描述仅供参考。即使你不符合职位描述中的所有技能和经验，也可以申请你喜欢的职位。可能你并不需要那些职位中的所有技能。

工程师在新公司工作一到六个月后开始面试候选人。新员工开始时通过观察资深面试官进行几轮面试，然后自己独立进行，这通常是他们得到的全部培训。面试官可能几个月前还处于你的位置，就像你一样，他们并不了解所有情况。即使经过多年的面试，我仍然担心在候选人面前出丑，给他们留下我公司的坏印象。

这种缺乏培训意味着即使在同一公司内，面试官们可能采用不同的面试技巧，对良好面试的看法也可能不同。用于评估候选人的评分标准——如果存在的话——是定性的而不是定量的（例如，“候选人表现出良好的调试技能”）。

招聘经理们对面试官的反馈的汇总方式也不同。有些招聘经理依赖于所有面试官的平均反馈。有些则依赖于最好的反馈——他们宁愿选择至少有一位面试官非常兴奋想要与之共事的人，而不是那些总体反馈良好但没有人特别热衷的人。谷歌就是一家重视热情推荐而非普遍冷淡评价的公司。

一些公司在积极扩张时，可能会雇佣任何没有理由不雇佣的人。而其他公司可能只有当有很好的理由雇佣某人时才会雇佣他们。

如果你认为某次面试表现不佳，不要绝望。除了你的实际表现之外，还有许多随机变量会影响面试的结果：提出的问题、面试官在你之前或之后看到的其他候选人、面试官的期望，甚至是面试官的心情。这绝不是你能力或自我价值的反映。公司也知道这一点，因此公司通常会在一年左右邀请被拒绝的候选人再次面试。
