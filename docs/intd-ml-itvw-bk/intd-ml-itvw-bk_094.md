# 8.2 深度学习架构和应用

> 原文：[`huyenchip.com/ml-interviews-book/contents/8.2-deep-learning-architectures-and-applications.html`](https://huyenchip.com/ml-interviews-book/contents/8.2-deep-learning-architectures-and-applications.html)

机器学习有三个主要子领域：语音和自然语言处理（NLP）、计算机视觉和强化学习。NLP 已经在商业智能、语音助手、机器翻译、自动完成和自动纠错、自动客户服务等应用中取得了成功。

计算机视觉是自动驾驶汽车的核心技术，在安全领域（如监控摄像头、人脸识别）、照片/视频生成（效果令人震惊地好）以及其他娱乐服务（如照片编辑、照片滤镜、换脸）等领域也有应用。

强化学习在部署上更为困难，因为现实世界的环境比模拟环境复杂得多，但我们已经看到了强化学习在广告竞价优化、无人机（如无人机）以及各种机器人应用（如仓库和生产机器人）中的使用。

一家公司或一个团队可能会专注于一个子领域。例如，苹果公司的 Siri 团队可能会专注于语音和自然语言理解，而特斯拉的 Autopilot 团队可能对计算机视觉更感兴趣。然而，一个子领域的技巧可以被用于另一个子领域，而且有些任务包含来自不同子领域的组件。无疑，成为你所在子领域的世界级专家具有价值，但要达到这个水平，你可能需要了解其他子领域的知识。
