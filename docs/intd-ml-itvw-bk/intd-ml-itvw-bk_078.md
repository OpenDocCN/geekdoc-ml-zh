# 5.2 概率和统计

> 原文：[`huyenchip.com/ml-interviews-book/contents/5.2-probability-and-statistics.html`](https://huyenchip.com/ml-interviews-book/contents/5.2-probability-and-statistics.html)

Reddit 上的一个用户曾经说过：“数据科学只是在 Mac 上做统计。”在机器学习和数据科学中，概率和统计的知识极为重要。如果你不理解交叉熵、KL 散度，或者只是普通的概率分布，那么机器学习中的大多数目标函数在统计学上可能几乎没有什么意义。
