# 2.3.1.1 背景 和 简历

> 原文：[`huyenchip.com/ml-interviews-book/contents/2.3.1.1-background-and-resume.html`](https://huyenchip.com/ml-interviews-book/contents/2.3.1.1-background-and-resume.html)

许多面试官通常会从“请告诉我关于你自己”这个问题开始。你应该想出一个不到一分钟的回答并在事先练习。你的回答应该是既个人的（你是你，没有人是别人）又专业的（为什么你适合这个角色）。你的回答可能会引导整个面试。

面试官也可能通过以下问题来测试你对一般专业水平的了解：“你在[机器学习|计算机视觉|自然语言处理|等]方面有多少参与？”不要吹牛。如果你假装比你实际经验更丰富，面试官会发现。

公司会想知道你的职业历史。他们可能会询问你的教育背景、以前的工作和过去的项目。你应该突出两点。一是你在每个职位上做出的有影响的决策，无论是积极的还是消极的。二是你为什么决定离开一家公司加入下一家。

面试官会试图看看你是否在你简历上声称的事情上有所行动。如果你有，你承担了多少责任。仅通过询问细节，就可以很容易地分辨出那些空谈者和那些实干者。他们可能会从你的简历中挑选一个项目，并让你解释你在其中做出的每一个选择。如果你声称熟悉 TensorFlow，那么你准备好讨论 TensorFlow 中的即时执行、tf.estimator 和分布式训练了。

雇主当然会想知道你的当前求职情况：你为什么在寻找工作，你在寻找什么。每个公司都会问的一个问题是，你是否有什么即将到来的出价/截止日期，他们应该知道。一些候选人可能会感到尴尬，承认他们还没有任何出价，担心公司会看轻他们。有些人会撒谎，告诉招聘人员他们有即将到来的现场面试或出价，尽管他们没有。

你可能可以侥幸逃脱——有些人甚至认为，由于公司经常利用他们的用户和员工，我们没有道德义务对他们诚实。然而，这是一个危险的斜坡。我不想成为那种为了成功而撒谎的人，但这是我们每个人必须为自己做出的个人决定。
