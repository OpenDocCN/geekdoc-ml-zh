# 游戏化面试流程

> [`huyenchip.com/ml-interviews-book/contents/0-gaming-the-interview-process.html`](https://huyenchip.com/ml-interviews-book/contents/0-gaming-the-interview-process.html)

人们经常问我：“你不担心候选人会只是记住这本书中的答案并操纵系统吗？”

首先，我不鼓励面试官询问本书中的确切问题，但我希望这本书能为面试官提供一个框架，以区分好问题和坏问题。

其次，只要这种记忆是有用的，记住某件事并没有什么错。问题在于当记忆变得不切实际时——候选人为了通过面试而记住某些东西，然后从未再次使用这些知识，或者不知道如何在实际情况中使用它们。

对于这本书，我的目标是只包含我认为以及许多有帮助的同事认为实用的概念。对于每一个概念，我都会问：“在现实世界中它在哪里被使用？”如果经过广泛的研究后我找不到一个很好的答案，那么这个概念就会被舍弃。例如，虽然我选择包含关于内积和外积的问题，但我省略了叉积。你可以在“不良问题”列表中看到被舍弃的问题，该列表位于[本书的 GitHub 仓库](https://github.com/chiphuyen)。这远非一个万无一失的过程。随着领域的扩展，现在不适用的一些概念可能会成为 2030 年 AI 研究人员一直在讨论的内容。

面试是有压力的，尤其是当它们是你梦寐以求的工作时。作为一个曾经站在你位置上的人，并且未来可能再次站在你位置上的人，我想告诉你，事情并不一定那么糟糕。每一次面试都是一次学习经历。一份工作邀请是很好的，但拒绝并不一定是坏事，也绝不是世界的末日。

影响面试结果的因素有很多随机变量：提出的问题，面试官在你之前或之后见过的其他候选人，面试官的期望，甚至是面试官的心情。这绝不是对你能力或自我价值的反映。

在我开始这个过程时，我几乎被申请的每一个工作都拒绝了。现在我只是以更低的频率被拒绝。继续学习和提高。你行得通！
