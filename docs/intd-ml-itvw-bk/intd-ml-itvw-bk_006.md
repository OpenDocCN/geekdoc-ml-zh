# 致谢

> 原文：[`huyenchip.com/ml-interviews-book/contents/0-acknowledgments.html`](https://huyenchip.com/ml-interviews-book/contents/0-acknowledgments.html)

没有众多朋友和同事的慷慨帮助，这项工作是不可能完成的。他们名单不分先后。如果我没有包括你的名字，那并不是因为你的贡献不重要，而是因为我的记忆有误。如果是这种情况，请告诉我，我会尽力纠正！

所有的荣耀都属于那些让它变得更好的人。所有的错误都属于我。
