# 2.3.1 行为问题

> 原文：[`huyenchip.com/ml-interviews-book/contents/2.3.1-behavioral-questions.html`](https://huyenchip.com/ml-interviews-book/contents/2.3.1-behavioral-questions.html)

每位面试官可能会先进行简短的自我介绍，然后提出一两个关于你背景的问题。这些问题旨在了解你是谁，也许还能让你感到更加自在。对于设有专门行为面试轮的公司，这部分在技术面试中应该比较简短。行为面试轮可能或可能不在午餐时间进行，通常由招聘经理或指定的行为面试官主持。

行为问题旨在评估你的价值观是否与公司的价值观相一致，以及公司是否能够为你提供一个让你能够茁壮成长的环境。这些问题可以分为四个主要主题：你的背景/简历、你的兴趣、你的沟通风格以及你的个性。
