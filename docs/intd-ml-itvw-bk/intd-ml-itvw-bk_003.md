# 关于问题

> 原文：[`huyenchip.com/ml-interviews-book/contents/0-about-the-questions.html`](https://huyenchip.com/ml-interviews-book/contents/0-about-the-questions.html)

本书中的问题是从数千个问题中精选出来的，其中大多数是在机器学习角色的实际面试中提出过的。你会发现其中有一些技术性错误或不明确的问题。这是故意的。有时，面试官会提出这些问题来观察候选人是否会纠正它们，指出边缘情况，或要求澄清。对于这些问题，附带的提示应该有助于澄清歧义或技术错误。

机器学习是一个工具，而要有效地使用任何工具，我们不仅要知道它是什么，还应该知道如何使用它、为什么使用它，或者何时使用它。因为“是什么”类型的问题很容易在网上找到，如果某样东西很容易获得，那么它就不值得测试。本书专注于“如何”、“为什么”和“何时”类型的问题。例如，与其询问 K-means 聚类的确切算法，不如询问在什么情况下 K-means 不起作用。你不需要理解 K-means 就能引用其定义，但你确实需要知道何时不应使用它。

尽管如此，本书仍包含少量“是什么”类型的问题。虽然它们不是好的面试问题，但它们对面试准备很有帮助。
