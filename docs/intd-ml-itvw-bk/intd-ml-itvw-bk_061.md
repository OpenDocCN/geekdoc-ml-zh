# 4.1 我需要多长时间来寻找工作？

> 原文：[`huyenchip.com/ml-interviews-book/contents/4.1-how-long-do-i-need-for-my-job-search.html`](https://huyenchip.com/ml-interviews-book/contents/4.1-how-long-do-i-need-for-my-job-search.html)

求职是一个漫长、压力山大，有时甚至令人沮丧的过程。时间表根据你所在的地方和你想去的地方而有所不同。在美国，你的求职应该在你希望开始新工作前三个月到六个月开始。对于学生来说，可能是在你学校最后一年开始。如果你在离开前需要提前通知——例如，在德国，离职员工必须提前三个月通知——这个过程应该开始得更早。如果你需要签证，你需要考虑获得签证所需的时间。

如果你计划在一年内找到工作，你可以在空闲时间回顾所有这些资源，并建立你的在线存在感。在回顾在线课程时，确保在观看讲座的同时完成作业。你可以尝试重新实现你感兴趣的论文，对其进行改进，并将它们发布在 GitHub 上。尝试至少参加几个 Kaggle 竞赛。

在你的面试前三个月，你可能有时间完成两到三门课程，并阅读三本书。对于课程，我推荐一门实践课程，如 fast.ai 的*面向编码者的实用深度学习*，以及一门理论课程，如 Coursera 的*机器学习*。你还可以查看我在斯坦福大学的课程：*[机器学习系统设计](http://cs329s.stanford.edu/)*，因为这门课程涵盖了生产中机器学习的实际挑战和解决方案。对于书籍，我推荐 Goodfellow 等人合著的*深度学习*，以及 Kevin P. Murphy 的*机器学习：概率视角*。

在你的面试前一周，回顾*[CS231N: 用于视觉识别的卷积神经网络](http://cs231n.stanford.edu/)*的笔记，特别是关于梯度下降、激活和优化的部分，以及重新观看*全栈深度学习*讲座，特别是关于*设置机器学习项目*和*基础设施和工具*的部分。你可能还想再次浏览这本书的第二部分和第三部分中的问题。你还应该回顾你以前的项目，以防你的面试官想了解所有关于它们的信息。

在你的面试前一天，确保你得到足够的睡眠。不要重复我熬夜复习并在半梦半醒的状态下参加面试的错误。提前 10 分钟到达，这样你就有时间安顿下来。

如果你得到了这份工作，但希望在将来找到更好的工作，这次要提前做好准备。如果你没有得到这份工作，就重新再来。

> ⚠ 三个月内如何成为机器学习专家 ⚠
> 
> 你不能。成为任何领域的专家都需要数年，甚至数十年的时间。远离那些声称能给你提供成为机器学习专家捷径的人。最坏的情况是，他们只会教你错误的机器学习知识。最糟糕的情况是，这甚至可能是一场骗局。
> 
> Peter Norvig，谷歌搜索部门的负责人，撰写了一篇精彩的博客文章，讲述了学习编程需要多长时间：[Teach Yourself Programming in Ten Years](https://norvig.com/21-days.html)。他的建议适用于机器学习。
