# 关于答案

> [原文链接](https://huyenchip.com/ml-interviews-book/contents/0-about-the-answers.html)

我开始写这本书时，怀有天真乐观的想法，以为我会为这本书中的每一个问题都写出答案。结果证明，在全职工作、教学任务、肆虐的疫情以及家庭成员住院的情况下，撰写 300 多个技术问题的详细答案是一项艰巨的任务。

考虑到我进展缓慢，我决定先发布草稿，然后继续写作/众包答案可能会更有效率。本书的第一稿涵盖了大约 10%的所有问题的答案。这些现有的答案，虽然我认为在数学上足够严谨，但更注重直观解释而非数学公式。问题的答案将会持续更新。通过[本书的 GitHub 仓库](https://github.com/chiphuyen)提交答案非常受欢迎。

招聘是一个过程，问题不是孤立评估的。你对每个问题的回答将作为你在整个过程中的表现的一部分来评估。一个声称从事计算机视觉工作但未能回答关于计算机视觉任务常用技术的问题的候选人，其评估结果将与完全不从事计算机视觉工作的候选人不同。面试官通常更关心你的方法，而不是你答案的实际目标正确性。
