# 1.1.2.2 研究科学家与研究工程师的比较

> 原文：[`huyenchip.com/ml-interviews-book/contents/1.1.2.2-research-scientist-vs.-research-engineer.html`](https://huyenchip.com/ml-interviews-book/contents/1.1.2.2-research-scientist-vs.-research-engineer.html)

对于研究工程师的角色，存在很多混淆。这是一个罕见角色，通常在行业中的主要研究实验室中可以看到。粗略地说，如果研究科学家的角色是提出原创想法，那么研究工程师的角色就是利用他们的工程技能为这些想法设置和运行实验。研究科学家角色通常需要博士学位和/或顶级会议的第一作者论文。研究工程师角色不需要，尽管发表论文总是有帮助。

对于某些团队来说，研究科学家和研究工程师之间没有区别。研究科学家首先应该是工程师。研究科学家和工程师都会提出想法并实施这些想法。研究员还可能担任顾问，指导研究工程师在其自身研究中的工作。研究科学家和研究工程师在论文中成为平等的贡献者并不少见。不同的职位名称主要是官僚主义的结果——研究科学家应该拥有更大的学术影响力，并且通常比研究工程师薪酬更高。

为了吸引人才，初创公司可能会在职位名称上更加慷慨。一位候选人告诉我，他选择了一家初创公司而不是一家 FAAAM 公司，因为初创公司给了他研究科学家的头衔，而那家大公司只给了他研究工程师的头衔。

Akihiro Matsukawa 在他的帖子《Research Engineering FAQs》中给出了关于研究科学家和研究工程师之间差异的有趣观点：[Research Engineering FAQs](https://mtskw.com/posts/re/)。
