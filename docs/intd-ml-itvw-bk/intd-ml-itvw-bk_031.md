# 2.1.1.4 初级与高级职位

> 原文：[`huyenchip.com/ml-interviews-book/contents/2.1.1.4-junior-vs-senior-roles.html`](https://huyenchip.com/ml-interviews-book/contents/2.1.1.4-junior-vs-senior-roles.html)

公司可能会将初级和高级职位放在不同的招聘流程中。那些没有通过先前工作经验证明自己的初级候选人可能需要经过更多的筛选。例如，亚马逊只为初级候选人要求编码挑战。然而，高级候选人可能会被问到更难的问题。例如，有几家公司告诉我，他们只向更高级的候选人提供机器学习系统设计问题^(21)，因为他们认为初级候选人可能没有足够的背景来很好地完成这些问题。

英伟达的人力资源经理告诉我：“*当你雇佣高级工程师时，你雇佣的是技能。当你雇佣初级工程师时，你雇佣的是态度*。”我曾在多个地方听到这种观点。

一家公共科技公司的高级总监告诉我，当他面试初级候选人，包括实习生和应届大学毕业生时，他更关心他们的思维方式、反应方式和适应能力。一个技术技能较弱但愿意学习的初级候选人可能会比技术技能较强但态度恶劣的另一个初级候选人更受欢迎。

^(21): 请参阅[机器学习系统设计](https://huyenchip.com/machine-learning-systems-design/exercises.html#exercises-rWl8SQW)（Chip Huyen, 2019）中的机器学习系统设计问题示例。
