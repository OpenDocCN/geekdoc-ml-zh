# 前言

> [`ml-science-book.com/preface.html`](https://ml-science-book.com/preface.html)

我们写这本书是因为我们对科学和机器学习充满热情，尤其是它们的相互作用。

我们的合作为博士研究期间开始，那时我们共同撰写了关于可解释机器学习的论文。然而，我们很快意识到可解释性只是谜题的一块。探索因果关系、不确定性量化、鲁棒性以及其他工具，证明是将“原始”机器学习转化为适当科学工具的关键。我们挖掘得越深，就越清楚，还有一个更大的部分缺失：需要强有力的理由来证明在科学建模中使用机器学习。所有我们的洞察力都引导我们写下了这本书。

准备开始一段科学哲学、机器学习理论、实用建模建议以及我们乌鸦科学家短篇故事的旅程。
