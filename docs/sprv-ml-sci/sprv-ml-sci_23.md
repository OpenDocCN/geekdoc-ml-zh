# 引用本书

> 原文：[`ml-science-book.com/citation.html`](https://ml-science-book.com/citation.html)

如果您觉得这本书对您的科研文章、博客文章或产品有所帮助，我们将不胜感激，如果您能引用这本书。您可以这样引用本书：

```py
Freiesleben, T. & Molnar, C. (2024). 
Supervised Machine Learning for Science:
How to stop worrying and love your black box.
https://ml-science-book.com/
```

或者使用以下 BibTex 条目：

```py
@book{freiesleben2024supervised,
  title    = {Supervised Machine Learning for Science},
  author   = {Timo Freiesleben and Christoph Molnar},
  year     = {2024},
  subtitle = {How to stop worrying and love your black box},
  url      = {https://ml-science-book.com/}
}
```

我们始终对人们如何在日常科学工作中使用机器学习感兴趣。如果您将我们的书作为参考文献，如果您能给我们写一行字并告诉我们用途，那将非常棒。这当然是可选的，只是为了满足我们的好奇心并激发有趣的交流。我们的邮箱是 science-book@christophmolnar.com
