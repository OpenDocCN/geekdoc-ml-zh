# 关于作者

> 原文：[`ml-science-book.com/about-the-authors.html`](https://ml-science-book.com/about-the-authors.html)

**克里斯托夫·莫拉尔**对与机器学习相关的一切都感兴趣。他是德国慕尼黑的独立研究员和图书作者。在这本书之前，克里斯托夫·莫拉尔已经写了[几本书](https://christophmolnar.com/books/)：可解释机器学习、建模心态、Python 中的符合性预测入门，以及使用 SHAP 解释机器学习模型。他还写了一个名为[Mindful Modeler](https://mindfulmodeler.substack.com/)的关于机器学习主题的时事通讯，这些主题超越了纯粹的预测。

在克里斯托夫成为独立作者之前，他在慕尼黑路德维希-马克西米利安大学（LMU）与蒂莫一起攻读关于可解释机器学习的博士学位。在获得博士学位、在 LMU 攻读统计学学士和硕士学位之间，克里斯托夫作为统计学家和数据科学家工作了几年。在业余时间，克里斯托夫喜欢骑自行车和做健美操，购买杂货，烹饪和享用美食，阅读刘慈欣和 qntm 的科幻小说，以及冰浴。你可以在[推特](https://twitter.com/ChristophMolnar)、[领英](https://www.linkedin.com/in/christoph-molnar/)和[个人网站](https://christophmolnar.com/)上找到他。

**蒂莫·弗雷伊斯莱本**对机器学习和它所引发的哲学问题真正感到好奇。蒂莫是一个科学哲学家和机器学习研究者的奇特混合体。他目前在图宾根大学的“机器学习为科学”集群担任博士后。以下是他的[网站](https://freieslebent.github.io/)和[谷歌学术个人资料](https://scholar.google.com.au/citations?user=BUvMXaAAAAAJ&hl=de)的链接。在业余时间，蒂莫喜欢徒步旅行、烹饪、阅读像陀思妥耶夫斯基和卡夫卡这样的奇怪书籍，从事像绘画和诗歌写作这样的创造性活动，以及足球、滑板和站立式划水等运动，和大多数人一样，喜欢与朋友和家人共度美好时光。

在他目前的位置之前，蒂莫在慕尼黑路德维希-马克西米利安大学（LMU）与克里斯托夫一起攻读博士学位，主要研究可解释机器学习、科学建模和因果关系。他在慕尼黑路德维希-马克西米利安大学和图宾根大学学习科学哲学、数学、机器学习和神经科学。蒂莫目前是卡尔·蔡司基金会项目“医疗保健中安全机器学习系统的认证和基础”的一部分，该项目也资助了他的博士后职位。
