# 致谢

本书灵感来源于[TinyML edX 课程](https://www.edx.org/certificates/professional-certificate/harvardx-tiny-machine-learning)和哈佛大学的 CS294r。它代表了与学生们、研究人员和从业者多年的合作，他们塑造了本书的发展。我们对那些开创性工作为其奠定基础的人们深表感激。

通过这次合作，我们对机器学习系统的理解加深了，我们意识到基本原理适用于所有规模，从小型嵌入式系统到大规模部署。这一认识促使本书从 TinyML 扩展到提供适用于所有机器学习系统实施规模的基石。
